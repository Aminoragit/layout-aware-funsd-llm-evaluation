# Real-Document Evaluation of Text-Only and Layout-Aware LLM Parsing on FUNSD

Reproducible preprint and rights-safe evidence bundle by Choi Seok Won.

- Paper: [`PAPER.pdf`](PAPER.pdf)
- Source manuscript: [`PAPER.md`](PAPER.md)
- Code, metrics, manifest, and license notice: [`artifacts/`](artifacts/)

The official FUNSD archive and raw OCR/model-output records are intentionally not redistributed. FUNSD users must obtain the dataset from its official source and accept its terms. See [`artifacts/DATASET_LICENSE_NOTICE.md`](artifacts/DATASET_LICENSE_NOTICE.md).

Set `PAPER2_LLM_URL` to an OpenAI-compatible endpoint before a live run. The reported model was `openai/qwable-3.6-27b-mtp`; private endpoint addresses are redacted.

Recommended arXiv classification: `cs.CV`, cross-list `cs.CL`.
